Describe three collision resolution strategies not mentioned here.
1. 2-choice hashing: Uses 2 hash functions to calculate the table location. The object is then place in the table location that contains fewer objects
2. Universal hashing: randomly select a hash function to calculate the table location and apply its own collision strategy
3. coalesced hashing: Similar to separate chaining, but it links chains of nodes within the table itself
